Spot-Hit Notifier
====================

Provides Spot-Hit integration for Symfony Notifier.

#### DSN example

```
// .env file
SPOTHIT_DSN=spothit://TOKEN@default?from=FROM
```

where:
 - `TOKEN` is your Spot-Hit API key
 - `FROM` is the custom sender (3-11 letters, default is a 5 digits phone number)
